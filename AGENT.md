# 项目协作约定

- 代码修改完成后，默认不要构建或打包安装包。
- 当用户输入 `BA` 时，表示需要执行 Android 打包；此时可以运行 `./gradlew :app:assembleDebug :app:assembleRelease` 这类 Android 打包命令，并按项目约定把 APK 复制到 `dist/` 目录。
- 当用户输入 `BI` 时，表示需要执行 iOS 打包；执行前先检查当前项目是否已有 iOS 打包配置和签名配置，再按项目约定打包。
- 日常开发完成后，优先使用轻量级校验，例如 `npm test` 和 JS 语法检查。
- 当用户输入 `GP` 时，表示需要提交当前改动，并推送到已配置的 git 远程仓库。
